Status of each service is defined by its progress along the docs, requests, models, mocks spectrum in the most recent released version.

docs
link to docs for service
Requests
all basic operations needed to support the shared api are implemented and tested
Models
service passes live, shared tests
Mocks
service passes mocked, shared tests

Completion is defined by progress along the n/a, pending, targeted, completed spectrum.

n/a
service not offered by provider.
pending
service is offered but is not yet completed
targeted
service is offered and in progress (see below for implementors)
completed
service is complete as per above
